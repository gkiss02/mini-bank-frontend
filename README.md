# MiniBank — Frontend

React + TypeScript single-page application implementing the MiniBank homework assignment.

## Requirements

- **Node.js** (developed and tested on **v24.13.0**)
- **npm**

## Getting started

After cloning the repository:

### 1. Install dependencies

```bash
cd mini-bank-frontend
npm install
```

### 2. Start the app in development mode

```bash
npm run dev
```

The app runs by default at `http://localhost:5173`.
